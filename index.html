Quick Reference Cheat Sheets
📦 B2C Pick Flow
  • 1Open HH app → Login with WH credentials
  • 2Go to Picking → Select B2C Picklist
  • 3Scan Picklist barcode or enter ID manually
  • 4App navigates to first shelf — go to location
  • 5Scan shelf barcode → scan item SKU → confirm qty
  • 6Repeat for all items → Mark picklist as Picked
🔄 Cycle Count Flow
  • 1Go to Inventory → Cycle Count on HH app
  • 2Select assigned count task or scan shelf barcode
  • 3Scan each item — app records vs. system qty
  • 4Discrepancies shown in red — confirm or flag
  • 5Mark count Complete → submitted for WH Manager review
  • 6WH Manager approves → inventory reconciled
🎯 Top 5 Objection Responses
  • 1"App is slow" → Check WiFi. Zebra TC21+ recommended. Offline mode available.
  • 2"Pickers won't learn it" → 15-min onboarding video. Pick flow is 6 taps — demo it live.
  • 3"We use paper" → Lead with error rate reduction data (avg 70% drop in mispicks).
  • 4"Devices not compatible" → App works on Android 8+. Share device compatibility matrix.
  • 5"Next quarter" → Offer 2-week pilot with 1 zone + dedicated support.

B2B Pick Flow Cheat Sheet
🏭 B2B Pick + HU Flow
  • 1Open HH app → Login → Go to B2B Picking
  • 2Select picklist → Activate Handling Unit
  • 3Scan carton barcode to map items to HU
  • 4Pick all items into HU — app validates qty
  • 5Seal and label HU → confirm completion
  • 6Move HU to dispatch staging area
🎯 B2B Objection Responses
  • 1"HU setup is complex" → One-time config. Template reuse after first setup. Demo it live.
  • 2"We bulk-pick manually" → HU-based picking is same speed, zero carton mismatch errors.
  • 3"Pickers prefer desktops" → 15-min training. Same flow they know — just on handheld.
